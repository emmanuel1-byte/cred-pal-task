# CRUD App with Authentication

This is a **CRUD (Create, Read, Update, Delete) application** with user authentication, built using **Express.js**, **TypeScript**, and **MongoDB**. The application is **Dockerized**, making it portable and easy to test.

## Features

* User authentication (signup & login)
* CRUD operations on resources
* Fully typed with TypeScript
* Portable using Docker

## Tech Stack

* **Backend:** Express.js + TypeScript
* **Database:** MongoDB
* **Containerization:** Docker

## Getting Started

### Prerequisites

* Docker & Docker Compose installed
* Node.js (if running locally)

### Run with Docker

```bash
docker-compose up --build
```

The app will be available at `http://localhost:3000`.

### API Documentation

Test all endpoints using Postman: [Postman Collection](https://documenter.getpostman.com/view/51129353/2sBXVo9oCT)
